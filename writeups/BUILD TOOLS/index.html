After development, applicaion needs to be deployed ona production server. To do that we need to package applicaion code and it's dependencies into a single movable file. Tha applicaion is called Artificat. packaging is also call building the code. Building the code → compiling → compressing. Artifacts files looks different depending on the programming language.

Install java and build tools

→ install java → Install maven → install node +name → install intellij

Build an artifact

→ We build artifacts using a build tool, every artifact is specific to the proamming lanaguage it was coded in. Install dependencies. compile and compress. Maven uses XML and Gradle uses groovy. Both have a command line tool and commands to execute the tasks.
