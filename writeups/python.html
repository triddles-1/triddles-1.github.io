PYTHON LISTS

Lists is an ordered collection of items, it is especially iseful when you need to work with a collecction of related items. It is a container that helps you store and manage pieces of data that can be different data types. lists are put inside a square bracket "[]". eg. shopping_list= ['flour', 'eggs', 'spinach', 'onion'] We can access the elements inside by getting their index. List count start from 0. To print the first item you type print(shopping_list[0]). You will get an output that displays "flour" on your terminal or IDE. Use -1 to access the last item on the list, -2 for the last but one , and so forth.
To add/append item to list, we use the append() argument. eg. shopping_list.append("bread"). now when you print your list you will notice that there is bread being added to list.
To remove item from in list, we can use the pop() argument. You type pop, then inside the parenthesis, you give it the item you want to remove. eg. shopping_list.pop(0) will remove flour from the list.
To slice list to get portion of the list, you just need to know the indexes of the item you want. then give the range with the ":" sign. Example, print(shopping_list[0:2]) will give you thr first two items in the list. NOTE: with slicing in python, the first indexis included in the slicing but te lat index is not.
To find the length of list, you need the len() argument. print(len(shopping_list))
